# coffee_shop_sales_sql_powerbi

 Performed end-to-end sales analysis using SQL by cleaning data, transforming date/time formats, and
creating KPIs such as Total Sales, Total Orders, and Quantity Sold.
◦ Wrote advanced SQL queries using JOINs, GROUP BY, window functions (LAG), and CASE statements
to calculate Month-over-Month (MoM) growth, weekday vs weekend sales, hourly and daily trends.
◦ Built an interactive Power BI dashboard to visualize sales trends, product category performance,
store-wise sales, and peak sales hours for business insights.
